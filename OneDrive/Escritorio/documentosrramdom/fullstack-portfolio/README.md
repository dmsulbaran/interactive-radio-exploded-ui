# Portafolio Web Premium: Desarrollador Full-Stack e IA

Este es un portafolio web moderno, responsivo y de alto rendimiento visual, diseñado específicamente para destacar habilidades en **Desarrollo Full-Stack (React, Node.js, SQL, MongoDB)** e **Integración de Inteligencia Artificial (Gemini API, Ingeniería de Prompts, Google Search API)**.

El sitio ha sido diseñado con un enfoque estético de nivel premium con variables CSS modernas, glassmorphism, fondos dinámicos interactivos y micro-interacciones pulidas sin depender de librerías de diseño pesadas.

---

## 🚀 Características Principales

1. **Diseño Estelar (Hero Section):** Gradiantes fluidos, tipografía de alta gama (`Outfit` & `Inter`), badges dinámicos y llamada a la acción atractiva.
2. **Consola de Simulación de IA (Terminal Interactiva):** Una terminal integrada en JavaScript que simula de forma interactiva entradas de prompt y respuestas generadas por Gemini integrando búsquedas web dinámicas.
3. **Fondo de Partículas Interactivo:** Un canvas HTML5 ultra-rápido que reacciona al movimiento del mouse del usuario atrayendo/repeliendo partículas.
4. **Galería de Proyectos Dinámica:** Filtros en tiempo real sin recargar la página para explorar trabajos específicos de Full-Stack o de Integraciones de IA.
5. **Formulario de Contacto Validado:** Entrada de datos con etiquetas flotantes interactivas, validaciones en tiempo real y retroalimentación animada de estado de envío.
6. **Responsividad Avanzada:** Estructuras fluidas optimizadas para dispositivos de 320px de ancho en adelante (Mobile, Tablet y Desktop) con menú móvil tipo hamburguesa adaptado.

---

## 🛠️ Stack Tecnológico Utilizado

- **Estructura:** HTML5 Semántico
- **Estilos:** CSS3 Moderno (CSS Grid, Flexbox, Variables CSS y Glassmorphism)
- **Lógica e Interactividad:** JavaScript ES6+ (Manipulación de DOM, Canvas API, Animaciones Temporizadas)
- **Fuentes:** Google Fonts (`Outfit`, `Inter`, `JetBrains Mono`)
- **Iconos:** SVGs en línea (Ultra-ligeros y vectoriales)

## 💻 Visualización Local

Para ver el portafolio en tu navegador:

1. Abre el archivo `index.html` directamente haciendo doble clic desde el gestor de archivos de tu sistema operativo.
2. O bien, inicia un servidor local ligero para una mejor experiencia de red:
   - Utilizando la extensión de VS Code **Live Server**.
   - O mediante terminal con `npx browser-sync start --server --files "*.*"` o `python -m http.server 8000`.
